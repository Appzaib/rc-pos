```php
  $this->extends theme.(startbootstrap-sb-admin);
```

### Description

RC-PoS is a clean, modern and fully responsive Point Of Sale Software built using Bootstrap 4 php and other technologies.


### Dependencies
* Bootstrap *4.1.3*
* Font Awesome *4.7.0*
* JQuery *3.3.1*
* Popper.js *1.14.4*
